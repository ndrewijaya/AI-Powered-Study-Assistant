# AI Tutor 🤖📚

Aplikasi asisten belajar interaktif berbasis web yang dibangun menggunakan **React** dan **JavaScript**. Aplikasi ini dirancang untuk membantu mahasiswa mengelola materi pembelajaran, merangkum teks, dan berinteraksi dengan fitur bertenaga AI.

---

## ✨ Fitur Utama

*   **Antarmuka Komponen (React):** UI yang reaktif, cepat, dan berbasis komponen modular yang rapi.
*   **Asisten Belajar AI:** Chatbot interaktif untuk membantu menjawab pertanyaan seputar materi kuliah.
*   **Ringkasan Otomatis:** Membantu merangkum artikel atau teks panjang dengan cepat.
*   **Desain Modern:** Desain UI/UX yang bersih, modern, dan nyaman digunakan di perangkat apa pun.

## 🛠️ Teknologi yang Digunakan

*   **Library Utama:** React.js (JavaScript)
*   **Styling:** CSS / Tailwind CSS (sesuaikan dengan yang kamu pakai)
*   **Tools:** Node.js & npm / Vite / Create React App

---
